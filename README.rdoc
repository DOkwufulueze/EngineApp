== demo

* Ruby version: ruby-2.2.1 [ x86_64 ]

* System dependencies: Check Gemfile

* Configuration: Install Contacts engine using the instructions in the engine's README file

* Database: This application runs on the MySQL server.
    Edit config/database.yml according to your system's peculiarity.
    Run migrations, including copying the Contact engine's migrations as stated in the engine's README file.
